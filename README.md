# ⚖️ Cherry Law (樱桃法律助手)

Cherry Law 是一款专为法律从业者打造的桌面级 AI 助手，深度优化法律业务场景的使用体验。它支持多个大语言模型（LLM）服务商，兼容 Windows、Mac 和 Linux 系统。

# 🌟 法律业务场景核心特性

1. **深度法律场景优化**：

- ⚖️ **法条检索与分析**：快速精准定位相关法律条文并进行智能解读。
- 📄 **法律文书助手**：辅助起草起诉状、答辩状、合同、法律意见书等各类文书。
- 🔍 **案情分析与预测**：输入案情摘要，AI 协助梳理法律要点并预测案件走向。
- 📚 **证据链梳理**：通过 AI 技术协助梳理复杂证据之间的逻辑关系。

2. **多样化 LLM 服务支持**：

- ☁️ 支持主流 LLM 云服务：OpenAI、Gemini、Anthropic、硅基流动等。
- 🔗 集成流行 AI Web 服务：Claude、Perplexity、Poe、腾讯元宝、知乎直答等。
- 💻 支持 Ollama、LM Studio 本地模型部署，确保敏感法律数据的私密性。

3. **文档与数据处理**：

- 📄 支持案卷扫描件、PDF、Word 等多种法律文书格式。
- ☁️ WebDAV 文件管理与数据加密备份，保障法律数据安全。
- 💻 法律术语与代码高亮显示。

4. **实用工具集成**：

- 🔍 全局搜索功能，快速定位历史案例与法律知识库。
- 📝 话题管理系统，按案件或课题分类管理对话。
- ⚙️ MCP(模型上下文协议) 服务，连接外部法律数据库。

5. **优质使用体验**：

- 🖥️ Windows、Mac、Linux 跨平台支持。
- 🎨 专业、克制的界面设计，支持明暗主题。
- 📝 完整的 Markdown 渲染，文书预览无忧。
